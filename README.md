# Evo-Leader: Эволюция Лидерства

Интерактивная платформа, исследующая трансформацию лидерства в Казахстане — от основателей Казахского ханства до современных лидеров.

## Особенности
- **Хронология лидеров**: Интерактивное путешествие во времени.
- **Интерактивная карта**: Ключевые регионы и личности.
- **Диалог с Мудрецом**: AI-чат на базе Gemini (жырау-стиль).
- **Путь Лидера**: Психологический квест на определение архетипа.

## Технологии
- React 19 + Vite
- Tailwind CSS 4
- Motion (Framer Motion)
- i18next (Kazakh/Russian)
- Google Gemini API

## Запуск локально

1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env` в корне проекта и добавьте ваш ключ:
   ```env
   VITE_GEMINI_API_KEY=ваш_ключ_здесь
   ```
4. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

## Деплой на Vercel

Проект полностью оптимизирован для деплоя на Vercel:

1. Импортируйте репозиторий в Vercel.
3. Нажмите **Deploy**.
